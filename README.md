# Sustainable Food Webshop

## Project Description
This project is a responsive sustainable food delivery webshop developed using HTML5, CSS3, and JavaScript. The website was created based on the provided Figma design and includes OpenAI API integration for chatbot functionality.

The project allows users to:
- Browse sustainable food products
- Explore responsive product layouts
- Interact with an AI chatbot assistant
- Experience a mobile-friendly accessible interface

---

# Features

- Responsive multi-page website
- Homepage, products page, and chatbot page
- OpenAI chatbot integration
- Async/await API handling
- Error handling and loading states
- Responsive design for mobile and desktop
- Semantic HTML structure
- Accessibility-focused layout
- GitHub version control
- Netlify deployment

- or another code editor.

3. Add Your OpenAI API Key

4. Run the Project

Run the project using:
